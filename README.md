# forgottodo
![forgottodo](https://user-images.githubusercontent.com/1820321/206865947-6cf8c73b-9935-4d38-a878-ca45835a2507.png)

Application to create tasks list  with an calendar system 






### Install the project
- Make sure of have nodeJS 10 or above installed
- Clone the project into you computer
- Run ```npm install``` to get the dependencies for the project

### Running
- then having sure of the dependencies are ready, run ```npm run android``` or ```npm run ios```
- Once the the metro compiler is running, in the same terminal type i for open iOS simulator. Or a for android emulator
